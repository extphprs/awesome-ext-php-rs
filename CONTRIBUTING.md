# Contributing

Thank you for helping grow this list!

## Adding a project

1. The project **must** use [ext-php-rs](https://github.com/extphprs/ext-php-rs) as a dependency.
2. Fork this repository and add your project to the appropriate category in `README.md`.
3. Use the format: `- [project-name](https://github.com/user/repo) - Short description ending with a period.`
4. Keep descriptions concise — one sentence, no marketing language.
5. Place your entry alphabetically within its category.
6. If no existing category fits, propose a new one — but prefer existing categories when possible.
7. Open a pull request using the provided template.

## Quality standards

- The project should have a README with usage instructions.
- The project should be functional (not just a skeleton or proof-of-concept).
- One project per pull request.

## Reporting broken links

If you notice a dead link, open an issue or a PR to remove/update it.
